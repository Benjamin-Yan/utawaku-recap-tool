- 一般來說，大多數流行歌曲的時間長度約在3分30秒到4分30秒之間 (我抓大的，約 270 sec)
- 未來預計做到的輸入格式(省去人工預處理的時間)
    ```
    00:00 抱きしめて
    05:09 Subtitle
    10:03 真夏の果実
    ```
- 目前暫時先用 CSV 的形式，如 `00:00,05:09,10:03`
- 輸入表示小時、分鐘和秒鐘的字串，可以是 "05:35"（省略小時）或 "1:05:35"（包含小時）
